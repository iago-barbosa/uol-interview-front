# ABORDAGEM PARA REALIZAÇÃO DO TESTE FRONTEND COMPASSO

Este projeto foi inicializado utilizando Reactjs com o comando [Create React App](https://github.com/facebook/create-react-app).

### `Inicializando a aplicação`

Para rodar essa aplicação no modo desenvolvimento, utilizar o comando "npm start" ou "yarn start".\
Utilize [http://localhost:3000](http://localhost:300) para vizualizar no navegador.

### `Bibliotecas utilizadas`

- Para navegação entre páginas foi utilizado: __react-router-dom__
  - Instalação: yarn add [react-router-dom](https://reactrouter.com/web/guides/quick-start)
  
- Para consumir os endpoints foi utilizado: __axios__
  - Instalação: yarn add [axios](https://github.com/axios/axios)
  
- Para facilitar a estilização da aplicação foi utilizado: __reactstrap__
  - Instalação: yarn add [reactstrap](https://reactstrap.github.io/)

- Para utilizar icones externos foi utilizado: __react-icons__
  - Instalação: yarn add [react-icons](https://react-icons.github.io/react-icons/)

- Para que Sass funcione na aplicação foi utilizado: __node-sass__
  - Instalação: npm install [node-sass](https://github.com/sass/node-sass)
